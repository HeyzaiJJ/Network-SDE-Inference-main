项目描述
复杂脑网络具有高度非线性与内禀随机性，传统建模依赖预设方程与简化假设，难以精准刻画 tau 蛋白病理扩散等真实动态，制约神经疾病机制解析与早期预测。依托脑区拓扑连接与病理时序数据，分离自驱动、节点交互与随机扩散三类动力学源，可从实验数据中无先验地推断可解释随机微分方程，精准还原脑网络时空演化规律，为脑疾病病理建模、动态预测与系统控制提供数据驱动的科学方法支撑。
